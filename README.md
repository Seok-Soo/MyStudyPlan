## 프로젝트 개요

시간별 일정 관리 앱

<br><br>

## 프로젝트 설명

MyStudyPlan은 자신의 공부량을 시간 단위로 기록하고 시각화하여 성취감을 높이고, 꾸준한 학습 습관 형성을 돕는 모바일 앱입니다.
